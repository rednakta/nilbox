# CLA Consent Log — Retroactive Relicensing

nilbox moved to a **dual-license** model (Community Edition under
[GPL-3.0](../LICENSE) + a separate [commercial license](../COMMERCIAL-LICENSE.md)) on
2026-07. Contributions made **before** the [CLA](../CLA.md) was in place were licensed
under GPL-3.0 only. To include those contributions under both licenses, we record each
contributor's explicit consent here.

This log is the due-diligence record for the project's copyright provenance. Do not edit
past entries once a consent link is recorded.

## How consent is obtained

1. Contact the contributor on the original PR (or a new issue) referencing this log.
2. Ask them to reply **from their own GitHub account** with the consent statement below.
3. Record the permalink to their reply in the table and set status to ✅ Consented.
4. If a contributor cannot be reached or declines, reimplement the affected code so no
   external copyright remains, and note that in the "Resolution" column instead.
